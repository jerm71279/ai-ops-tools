# Phase 2: Recommendation & Design

1. Select core router/firewall based on throughput, features, and budget requirements

2. Design VLAN and IP addressing scheme with standard VLANs (10, 20, 30, 40, 50)

3. Plan wireless network architecture including SSID naming, security, and AP coverage map

4. Design security policies including firewall rules, access control lists, guest portal, and VPN requirements

5. Plan redundancy and failover including dual WAN, switch stacking, and power redundancy

6. Create proposed network topology diagram showing all devices, VLAN segmentation, and connections

7. Conduct customer design review meeting to present topology, VLAN design, and security policies

8. Obtain customer design approval with signed Design Approval Document

9. Develop naming conventions for devices using Location-Type-Number format

10. Create quote in OneBill including equipment, licensing, labor, and installation timeline
