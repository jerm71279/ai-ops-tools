# Phase 3: Pre-Configuration & Preparation

1. Complete equipment procurement and add to SmartSheets as "Assigned/Pending Install"

2. Communicate with carrier to obtain ECD, FOC, IP information, and update Circuit Inventory

3. Create configuration templates for firewall, router, switches, and wireless controller

4. Configure all devices in lab environment with management IPs, VLANs, security policies, and SSIDs

5. Conduct lab testing: internet connectivity, VLAN routing, wireless SSIDs, security rules, and VPN

6. Create configuration backups and store in password vault/SharePoint

7. Complete Pre-Config Test Report and obtain Engineering Lead approval

8. Schedule installation with customer, update calendars, and complete Installation Readiness Checklist

9. Conduct customer pre-install briefing to confirm timeline, downtime window, and expectations

10. Day before install: review checklist, confirm equipment loaded, team ready, and site access verified
