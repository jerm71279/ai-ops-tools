## Graph View Controls

### Filters Panel (Right Side)
Click **"Filters"** to access:

**Search:**
- Type keywords to highlight matching notes
- Example: "policy" shows all policy docs

**Tags:**
- Filter by tags like `#it-services`, `#engineering`

**Groups:**
Create color-coded groups:
1. Click "+ New group"
2. Enter search term (e.g., "sales")
3. Pick a color
4. All notes matching that term will be colored

### Display Panel (Right Side)
Adjust visualization:

- **Node size**: Make important notes bigger
- **Link distance**: Spread out or cluster nodes
- **Repel force**: Push nodes apart
- **Link thickness**: Make connections more/less visible
- **Fade threshold**: Fade out distant connections

### Useful Settings to Try

**View only one team:**
1. Click **Filters**
2. In "Search" box, type team name (e.g., "engineering")
3. Check "Show orphans" OFF to hide unconnected notes

**See hub notes:**
